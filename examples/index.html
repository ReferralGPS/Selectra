Selectra

v1.1.1

Alpine.js + Tailwind CSS component. No jQuery required. Fully accessible and customizable.

Progressive Enhancement Use native <select> tags with x-selectra. It falls back to a standard select when JavaScript is disabled or hasn't loaded.
